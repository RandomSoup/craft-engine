# craft-engine
Fork of the original [craft](https://github.com/code-dot-org/craft) by [Code.org](https://code.org).

## Building
To build craft-engine, do `make` in its directory.

## Usage
To start a simple server, do:
```sh
cd ./build/
python3 -m http.server
```
